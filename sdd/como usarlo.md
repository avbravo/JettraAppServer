
### Cómo usar este `brain.md` con Gemini / Antigravity IDE

Cuando vayas a iniciar una tarea de expansión del framework (por ejemplo, añadir un nuevo componente visual o un interceptor de seguridad), puedes abrir tu sesión con el siguiente prompt de contexto:

> *"Actúa como un Arquitecto de Software Experto en Java 25. Analiza el archivo de especificación de desarrollo `brain.md` adjunto. Basándote en las reglas de Spec-Driven Development establecidas para los módulos **JettraFlux** y **JettraFluxExample**, necesito que diseñes el nuevo componente... [especificar tarea]"*

De esta forma, la IA sabrá exactamente a qué repositorios afecta el cambio, qué tecnologías tiene prohibido usar (como JavaScript/HTML nativo en las vistas o jQuery) y qué arquitectura de alto rendimiento debe respetar obligatoriamente.
